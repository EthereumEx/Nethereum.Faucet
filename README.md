# Nethereum.Faucet

Simple faucet to fund Ethereum accounts

##Configuration settings:
### Appsettings.json

```
	"EthereumAddress": "http://ETHEREUM-ADDRESS:8545",
    "FunderPrivateKey": "PRIVATE KEY",
    "MaxAmountToFund": 100,
    "AmountToFund": 10,
    "UrlTxnExplorer": "http://BLOCKEXPLORER-ADDRESS/tx/"
```
